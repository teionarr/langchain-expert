# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be decent. Critique code and claims, not people. Disagreement backed by evidence is the point
of this project — a rubric that can't be challenged isn't measuring anything.

Report unacceptable behaviour by opening an issue or contacting the maintainer.
